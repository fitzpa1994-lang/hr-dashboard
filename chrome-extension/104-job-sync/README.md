# SPORTON 104 職缺同步掛件

此掛件提供兩個由招募作業台手動觸發的功能：

- 「從 104 更新職缺」：讀取已登入的 104 招募管理「所有職務」頁。
- 「擷取條件」：讀取最近使用的 104 查詢人才頁；若仍停在條件頁，會執行一次查詢並儲存可重現的搜尋網址。

同步完成後，刊登快照會由招募作業台保存，並在「職缺總覽」中與內部缺額資料進行人工確認配對；掛件本身仍不會改寫內部職缺、缺額或到職扣缺規則。

自 v1.3.0 起使用同步 contract v2。掛件只有在能確認目前為未帶篩選條件的「所有職務」、每頁表頭與總筆數一致，且每筆刊登狀態都可識別時，才會送出完整快照；已明確顯示「共 0 筆」時也會保存為有效的空快照。任何一項無法確認都會取消更新並保留原資料。更新掛件後請在 `chrome://extensions` 重新載入，再重新整理招募作業台。

v1.3.1 相容 104 最新的浮動表頭結構（`thead` 直接包含欄位 `th`），並新增由面板主動確認掛件版本的握手；欄位數量、狀態欄或分頁資訊不一致時仍會安全拒絕同步。

## 安裝

1. 開啟 `chrome://extensions`。
2. 開啟右上角「開發人員模式」。
3. 點「載入未封裝項目」。
4. 選擇此資料夾：`chrome-extension/104-job-sync`。
5. 重新整理 `https://sp-hr.zeabur.app/#talent-search`。

掛件不會定時執行，也不會保存或上傳 104 帳號密碼。只有按下招募作業台上的同步或擷取按鍵才會運作；搜尋條件只儲存在目前 Chrome 的招募作業台網站資料中。
